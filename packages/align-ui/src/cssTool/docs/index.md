# 样式系统

一款为组件开发场景下在 RN 端使用的 css 样式书写方式，一起为开发者提供方便快捷的开发体验

## 特性

主要具备以下特性：

- 🚀 **轻量级的应用**
- 🚥 **约定式样式增强**
- 💎 **为组件研发而生**

## 属性支持

### Key

- ✅ = Implemented, exists in RN
- 😎 = Implemented, but is RN ONLY
- 🚨 = not implemented
