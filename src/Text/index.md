# Text 文本

支持字号、字重、颜色、常用 苹果 字体等，封装行内占位，三端差异抹平等等常用功能。

### 何时使用

- 当你需要展示文本的时候

### Text

#### 属性

组件继承 [`Text`](https://reactnative.dev/docs/text)

| 属性           | 说明       | 类型      | 默认值 |
| -------------- | ---------- | --------- | ------ |
| containerClass | 样式简写   | `string`  | -      |
| style          | 单独的样式 | `Object`  | -      |
| children       | 被包裹元素 | `Element` | -      |
